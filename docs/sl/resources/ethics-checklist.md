# Etični in pravni kontrolni seznam

Uporabite ga pred zbiranjem, zajemanjem, anotiranjem, kartiranjem, modeliranjem, objavo ali uporabo UI na kulturnem gradivu. Odkljukano polje ni etična odobritev; vsak odgovor podprite s kratko zabeleženo odločitvijo.

## Namen in nujnost

- Je raziskovalni ali učni namen izrecen?
- Je vsako zbrano polje potrebno za ta namen?
- Bi lahko vprašanje rešili z manj občutljivimi ali manj določljivimi podatki?
- Ali prihodnjo raziskovalno rabo pošteno predstavljate kot raziskovalno in ne kot neomejeno dovoljenje?

## Ljudje in skupnosti

- So živi ljudje predstavljeni neposredno ali posredno?
- Bi povezovanje zbirk razkrilo družinske, zdravstvene, politične, verske, migracijske ali identitetne podatke?
- So vključeni mladoletni, ranljivi ljudje ali majhne skupnosti?
- Je soglasje potrebno, mogoče in vsebinsko smiselno?
- Bi anonimizacija res preprečila ponovno identifikacijo po združevanju ali navajanju?
- Ima predstavljena skupnost dejansko možnost komentirati, omejiti ali popraviti?

## Viri in pričakovanja

- Je bilo gradivo ustvarjeno za javno kroženje ali omejen kontekst?
- Se tehnična dostopnost ujema z razumnimi pričakovanji o ponovni uporabi?
- So izbrisani, omejeni ali nadzorovani viri obravnavani primerno?
- Ali množični zajem spremeni etični značaj posamezno vidnih zapisov?

## Pravice in licence

- Kdo ima pravice do besedila, slike, zvoka, videa, kode, anotacije in podatkov?
- Kaj posamezna licenca dovoljuje: analizo, navajanje, razširjanje, prilagoditev ali komercialno rabo?
- So pomembni pogoji platforme, pravice do zbirke ali pogodbene omejitve?
- Je mogoče deliti surove podatke ali le metapodatke, izpeljane značilke, identifikatorje oziroma skripte za rekonstrukcijo?
- So navedbe avtorstva in obvestila o pravicah tretjih oseb popolna?

## Reprezentacija in pristranskost

- Kaj ni bilo nikoli ustvarjeno, ohranjeno, digitalizirano, katalogizirano ali strojno berljivo?
- Katere skupine, jeziki, obdobja ali žanri so premočno oziroma premalo zastopani?
- Se kakovost OCR, anotacije ali modela razlikuje med pomembnimi skupinami?
- Bi kategorije lahko ponavljale škodljive zgodovinske klasifikacije?
- So manjkajoči podatki jasno ločeni od odsotnosti?

## UI in avtomatizacija

- Je naloga primerna za avtomatizacijo ali zahteva človeško kontekstualno presojo?
- So model, različica/datum, poziv, parametri in podani kontekst zabeleženi?
- So generirane dejanske trditve in navedki preverjeni ob virih?
- Obstaja izhodišče ali ročno označen vzorec?
- So pregledane napake po skupinah in nestabilni rezultati?
- Je imenovana oseba, odgovorna za končne odločitve?

## Objava in upravljanje

- Je javna objava potrebna ali bi bil omejeni dostop varnejši?
- So negotovost, omejitve in uporaba modela vidni bralcem?
- Obstaja pot za stik, popravek in umik?
- So sodelavci navedeni in so zavestno sprejeli licenco?
- Sta ocenjevanje študentov in soglasje k javnemu avtorstvu ločena?
- Je določen datum ponovnega pregleda programsko odvisnega ali občutljivega gradiva?
