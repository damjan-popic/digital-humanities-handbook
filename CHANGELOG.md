# Changelog

All notable changes are documented here. The project follows semantic versioning for published releases.

## [Unreleased]

### Added

- Added four paired chapters on digital-humanities histories, models and evidence, critical infrastructures, and the Slovenian ecosystem.

### Changed

- Introduced the warm editorial theme together with contrast, keyboard-focus and responsive accessibility fixes.
- Organized the stable handbook core into four parts covering contexts, research design, analytical methods and responsible publication.
- Revised both paired learning paths to integrate the expanded core while preserving their fourteen-module course designs.
- Structured the bilingual review manuscripts by orientation and part, and strengthened checks for navigation parity, duplication and order drift.

## [0.1.0] - 2026-07-20

### Added

- Reframed the project as the **Digital Humanities Handbook**.
- Added bilingual MkDocs architecture with English and Slovene editions and contextual language switching.
- Added a peer-reviewable handbook core and two course learning paths.
- Added editorial, review, translation, versioning and contribution policies.
- Preserved and reclassified the practical workflow and case-study library from the Digital Linguistics Playbook.
- Added publication-planning materials for Znanstvena založba FF.
