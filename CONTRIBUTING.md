# Contributing

Contributions from students, researchers, librarians, developers and cultural-heritage practitioners are welcome.

## Good first contributions

- correct a factual or technical error;
- improve a validation step or expected output;
- translate or review one page;
- add a small reusable dataset with clear rights information;
- document one bounded workflow;
- add a disciplinary case study that explains both method and limits.

## Before opening a pull request

```bash
python scripts/check_handbook.py
python scripts/check_translation_coverage.py
mkdocs build --strict
```

A contribution must state what changed, how it was checked, which language/version it affects, and any rights or privacy constraints.
